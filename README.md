<h1>Java Factory Pattern (creational)</h1>

<p><b>Concepts:</b></br>
- Doesn't expose instantiation logic</br>
- Defer to subclasses</br>
- Common interface</br>
- Specified by architecture, implemented by user</p>

<p><b>Examples in Java:</b></br>
- Calendar</br>
- ResourceBundle</br>
- NumberFormat</p>
